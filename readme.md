**Swashbuckler** Diary -> Swashbuckler-ize-er -> Swashbucklerizer  

> 这构词法笑死我了

这个小脚本最初是在期末的考试周的前一个周末搓出来的，所以首先注释几乎没有，其次还写成一坨ww
